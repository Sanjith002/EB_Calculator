# ⚡ TNEB Electricity Bill Calculator

A full-stack Tamil Nadu Electricity Bill Calculator built with React, Django REST Framework, and Tailwind CSS. Calculates electricity bills using the latest TNEB tariff slabs with a modern responsive user interface.

---

## 🌐 Live Demo

**LINK:** https://your-live-demo-link.com/

---

## 🚀 Features

* Calculate Electricity Bill
* Latest TNEB Tariff Calculation
* Slab-wise Bill Breakdown
* Instant Bill Estimation
* Input Validation
* Error Handling
* Responsive UI
* REST API Integration

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Axios
* Tailwind CSS
* TanStack React Query
* Framer Motion
* React Icons
* React Hot Toast

### Backend

* Django
* Django REST Framework
* django-cors-headers
